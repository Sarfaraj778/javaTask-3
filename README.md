# Assignment 03 - Even Odd Checker

##  Project Overview

This project is a simple Java application that determines whether a given integer is even or odd using the modulo (%) operator. It accepts user input through the Scanner class and displays the result.

---

##  Objectives

- Understand the modulo (%) operator.
- Learn conditional statements (if-else).
- Accept user input using Scanner.
- Display meaningful output.

---

##  Technologies Used

- VS Code
- Git & GitHub

##  How to Run

Compile the program:

```
javac EvenOddChecker.java
```

Run the program:

```
java EvenOddChecker
```

---

##  Sample Output

```
Enter a number: 8
8 is Even
```

```
Enter a number: 15
15 is Odd
```
